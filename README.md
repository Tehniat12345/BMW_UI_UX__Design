# Automotive Hero Banner Website

A premium, fully responsive landing page hero banner built using clean HTML5 and CSS3. The design is inspired by modern, luxury, and high-performance automotive websites.

## 🚀 Live Demo & Preview
* **Desktop Layout**: Split-screen design with text on the left and a high-quality sports car image on the right.
* **Theme**: Deep matte black and grey palette with a vibrant royal blue Call-To-Action (CTA) button.

## 🛠️ Tech Stack Used
* **HTML5**: Structured semantic markup for content, headings, and buttons.
* **CSS3**: Custom styling, flexbox/grid layout, hover animations, and custom font styling.

## ✨ Key Features
* **Modern Typography**: Bold title paired with a unique outlined text effect for "ADRENALINE".
* **Responsive Layout**: Adapts perfectly across mobile, tablet, and desktop screens.
* **Interactive Elements**: Smooth hover and active states on the "BUILD YOUR OWN" button.
* **Performance Optimized**: Lightweight code ensuring fast loading times.

## 📂 Project Structure
```text
├── index.html        # Main HTML file for the webpage
```

## ⚙️ How to Run Locally
1. **Clone or Download** this repository to your local machine.
2. Ensure your car image is placed inside the `assets/` folder and correctly linked in your files.
3. Open `index.html` in any modern web browser (Chrome, Edge, Safari, Firefox).
